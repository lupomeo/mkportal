Comment mettre a jour MKPortal avec Oxygen
 

Attention

  • Ceci est seulement pour une mise à jour, Mkportal M08 est requis pour cette mise à jour!!
  • M08 est requis. Vous ne pouvez pas mettre à jour depuis une autre version de Mkportal.


Pour mettre à jour MKPortal M08 à M09 vous devez:

  1. Envoyez les fichiers de ce pack dans votre site en reécrivant sur tout les fichiers EXCEPTE mkportal/conf_mk.php.
    (vous devez reécrire également le fichier index.php de MKportal).
     
  2. Vous pouvez commencer Mise à jour en tapant cet adresse:
    http://www.votresite.com/mkportal/upgrade.php
  3. Supprimer le fichier mk_install.php du serveur

    4. Mise à jour complete.
