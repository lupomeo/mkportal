How to Upgrade MKPortal for Phpbb
 

Please note:

  • This is only an upgrade, Mkportal M08 is required for this upgrade!!
  • M08 is required. You cannot upgrade from other MKportal version.


To upgrade your version of mkportal from M08 to M09 you have to:

  1. Upload the files contained in this package int your site overwriting all the old files EXCEPT the file mkportal/conf_mk.php.
    (you need to overwrite also the file index.php of MKportal).
     
  2. Now you can start upgrade script typing the url:
    http://www.yoursite.com/mkportal/upgrade.php
  3. Remove file mkportal/upgrade.php from your server

    4. Upgrade completed.
