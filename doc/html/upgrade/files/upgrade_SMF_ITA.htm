Come Aggiornare MKPortal per SMF
 

Attenzione:

  • Questo è solo un upgrade!! E' necessario aver installato già MKportal M 0.8
  • Con queste istruzioni non è possibile aggiornare da versioni precedenti alla M 0.8.


Per aggiornare la tua versione di MKPortal dalla M 0.8 alla M 0.9 occorre:

  1. Trasferire i file contenuti in questo pacchetto nel tuo spazio web sovrascrivendo tutti i file eccezion fatta per mkportal/conf_mk.php.
    (è necessario anche sovrascrivere l'index.php di MKportal).
     
  2. Avviare lo script d'upgrade digitando l'url:
    http://www.tuosito.it/mkportal/upgrade.php
  3. Cancellare il file mkportal/upgrade.php dal server

Upgrade completato.
