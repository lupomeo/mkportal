Comment installer MKPortal avec SMF

Attention:

  • SMF 1.x est requis pour installer cette version de MKPortal.
  • Il est recommandé de sauvegarder sa base de données avant d'installer ce produit.
     


L'installation de MKPortal est très simple :

  1. La pack d'installation (fichier zip) contient le dossier "upload". Vous devez transférer le dossier "upload" sur votre espace web
    (pour exemple http://www.votresite.com/) dois être de ce type:

    /forum          (ou un autre nom de dossier qui contient toute l'installation de SMF)
    /mkportal     (contenue dans le dossier "upload")
    /index.php   (inclu dans le pack mkportal)

    Vous devez obtenir cette structure:

    http://www.votresite.com/forum
    http://www.votresite.com/mkportal
    http://www.votresite.com/index.php       (index de mkportal)

    il y a d'autre possiblité si vous gardez le nom de dossier mkportal
    Si vous avez l'index de votre forum sur la racine du serveur vous devez la déplacer dans un sous-dossier
    (vu dans l'example ci-dessus).


     
  2. Vous pouvez commencer l'installation en tapant cet adresse:
    http://www.votresite.com/mkportal/mk_install.php

    Suivez juste les instructions et c'est fait


     
  3. Donnez les permissions d'écriture (chmod 777) a ces fichiers et leurs contenues:

    mkportal/conf_mk.php
    mkportal/cache
    mkportal/blog
    mkportal/modules/downloads/file
    mkportal/modules/gallery/album
    mkportal/modules/reviews/images


     
  4. Supprimer le fichier mk_install.php du serveur

Installation terminé!!
Amusez-vous bien !!!

Si vous voulez intégrer SMF à MKPortal, vous devez:

  • Allez dans le Panel Admin Mkportal -> dans vos préférences cocher l'option "Visualisation du Forum : À l'Interieur de MKPortal"

et modifiez ce fichier (faire une sauvegarde!):
